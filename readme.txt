== Description ==
This plugin allows you to only give access to certain files or folders to people that have purchased a membership in Restrict Content Pro.

For more information - please check out this page: https://github.com/angay9/rcp-file-protector.

Please check out this tutorial to find out how to use the plugin:
https://www.youtube.com/watch?v=pFC9PP34818

== Installation ==
Simply find this plugin in Wordpress Plugin Directory and install it like you would any other plugin.
